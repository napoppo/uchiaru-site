利用者入力データ 在庫・買うもの・カテゴリ・置き場所・単位・お店・メモ・賞味期限・数量・添付画像 等 アプリ内入力 本アプリの機能提供（一覧表示・編集・通知 等）

3. 利用する外部サービス・送信先

本アプリは以下の外部サービスを利用しており、利用者情報の一部が各社に送信されます。各社における取扱いは各社のプライバシーポリシーに従います。

Firebase (Google LLC)

  • Cloud Firestore / Firebase Storage: 入力データ・画像の保存と同期
  • Firebase Authentication: ログイン認証 (Apple / Google / メール)
  • Firebase Cloud Functions: 通知配信・バックアップ・課金 Webhook 処理
  • Firebase Cloud Messaging: プッシュ通知配信

Firebase プライバシーポリシー

RevenueCat

アプリ内課金 (Pro プラン) の管理に利用します。匿名 ID と購入トランザクション情報が送信されます。
RevenueCat Privacy Policy

Google AdMob (無料利用時のみ)

無料プラン利用時にバナー広告を表示します。広告配信のため広告 ID 等が Google に送信されます。iOS では、配信前に App Tracking Transparency (ATT) の許可ダイアログを表示し、利用者の選択に従います。許可しなくても本アプリは利用できます (広告がパーソナライズされなくなります)。
Google 広告ポリシー

楽天商品検索 API (Rakuten Group, Inc.)

在庫追加時の「バーコードから商品を取得」機能でのみ、スキャンした JAN コードが楽天株式会社のサーバーに送信されます。利用者を特定する情報 (uid・メール等) は送信しません。
楽天プライバシーポリシー

Apple / Google (プラットフォーム)

サブスクリプションの決済・継続管理は App Store / Google Play を介して行われます。決済情報は本アプリ運営者には共有されず、各プラットフォームのポリシーに従って取り扱われます。

4. 共有グループ内での情報共有

利用者が共有グループに参加すると、そのグループに紐づく在庫・買うもの・カテゴリ・置き場所・単位・お店・メモ・操作履歴・グループ情報は、同じグループのメンバー全員に表示・編集されます。グループから退出するか、グループが削除されるまで、これらの情報はメンバー間で共有され続けます。

招待コードを知っている第三者は、有効期限内であれば共有グループに参加できます。コードの取り扱いには十分ご注意ください。

5. 第三者提供

以下のいずれかに該当する場合を除き、利用者情報を第三者に提供しません。

  • 利用者本人の同意がある場合
  • 法令に基づく場合 (裁判所・警察等からの正当な開示請求)
  • 本アプリの提供に必要な外部サービスへの委託 (上記「3. 利用する外部サービス」)
  • 利用者の生命・身体・財産の保護のために必要かつ本人の同意取得が困難な場合

6. 保存期間

  • 利用者の入力データ・共有データ: アカウントまたはデータが削除されるまで
  • 操作履歴: 無料プランは最新 20 件まで / Pro プランは 30 日間 (最低 100 件)
  • バックアップ (Pro): 30 日間自動保持。30 日を超えたものは Cloud Functions により自動削除
  • 購入情報: 法令により必要な期間 (税務上の保存義務等)

7. 利用者の権利

利用者は、自己の情報について次の権利を行使できます。

  • 開示請求: アプリ内の各画面、操作履歴、設定 → アカウントから確認できます。
  • 訂正・追加: アプリ内の編集機能でいつでも行えます。
  • 削除: アプリ内の「アカウント削除」より、利用者本人のアカウント・所属するグループ (オーナーの場合) を削除できます。データは合理的な期間内に完全削除されます。
  • 利用停止: アプリを削除することで本アプリへの情報提供は停止できます。サーバー上のデータも削除したい場合は前項のアカウント削除を行ってください。

8. 安全管理措置

  • 通信は HTTPS / TLS により暗号化されます。
  • Firestore セキュリティルールにより、原則として同じグループのメンバー以外はデータにアクセスできません。
  • 認証は Firebase Authentication を使用し、本アプリ運営者がパスワードを保有することはありません。

9. 子どもの利用

本アプリは 13 歳未満の利用を想定していません。保護者の同意なく 13 歳未満の個人情報が登録されていることが判明した場合、当該情報を削除します。

10. クッキー類似技術

本アプリはブラウザ Cookie を使用しませんが、ログイン状態を維持するために Firebase Authentication のトークンを端末内に保存します。トークンはサインアウト時に削除されます。

11. 本ポリシーの変更

本ポリシーは必要に応じて変更することがあります。重要な変更を行う場合は、アプリ内通知または本ページの掲載によりお知らせします。
